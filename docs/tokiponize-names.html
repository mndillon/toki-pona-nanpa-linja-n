Tokiponize Proper English Names

Best-effort phonetic Tokiponization for English proper names. Output is guaranteed to be valid CV(N) syllables (final n only). o ante e nimi jan tawa toki pona. nimi li ken jo e kipisi CV(N) taso.

This is a best-effort approximation. Pronunciations vary, and different Tokiponization choices can be valid. Please review and adjust the result to match your intended pronunciation and community conventions. sona: ilo ni li pana e nimi pi ken mute taso. toki uta li ken ante lon jan ante. o lukin e nimi ni, o ante e ona tawa toki uta sina.

Accepts spaces, hyphens, apostrophes, and common patterns like McX, O’X. Prefix a token with / for direct IPA input (e.g. /dʒan/); wrap multiple words like /dʒan ɫuk/.
Style notes: Different tokiponizations can be valid. If you prefer a different style, adjust Mapping policy (phoneme→TP choices like r→w/l/∅) and use Overrides for specific names.
Keep final consonants = more recognizable (loanword-like). Off = stricter (drop illegal finals).

Custom Name Overrides

        
      

      

        

          Edit your custom name overrides as JSON. Keys are English names or phrases (case-insensitive),
          values are Tokiponized forms (without "jan " prefix).
          Built-in defaults are not shown here — only your custom additions.
        

        

          Example:
          
{
  "sinead": "sineta",
  "ciarán": "sijawan",
  "old macdonald": "makitonapo"
}

          Keys match whole input lines or individual tokens, case-insensitively.
          Values should be valid Tokiponized syllables (no "jan " prefix needed).
        

        
        
        

          
          
        

        

          Built-in defaults remain active regardless of what you put here.
          Removing a key reverts that name to automatic Tokiponization.

IPA Token Overrides

        
      

      

        

          Edit your IPA token overrides directly as JSON. Keys are the English token (normalised to lowercase),
          values are IPA strings that replace the auto-generated pronunciation for that token.
          Changes here are equivalent to editing via the table editor.
        

        

          Example:
          
{
  "jean": "ʒdan",
  "sinead": "ʃɪneɪd",
  "niamh": "niːv"
}

          Keys are the plain English token in lowercase. Values are IPA symbols —
          use the IPA editor (click any underlined segment in the results table) to look up the right symbols.
        

        
        
        

          
          
        

        

          Removing a key clears that token's override and reverts to auto-generated IPA.
