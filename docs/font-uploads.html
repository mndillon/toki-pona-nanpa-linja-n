Font Uploader and nanpa-linja-n cartouche companion font creator

Upload a UCSUR-compatible font, create or replace its nanpa-linja-n cartouche companion font, preview cartouches, and save the font pair so it can be selected on other pages of this site after those pages are refreshed. (Currently supports *.ttf and *.otf format fonts. The font must define the UCSUR codepoints U+F1990, START OF CARTOUCHE, and U+F1991, END OF CARTOUCHE.)
